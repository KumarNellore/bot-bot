This is a Selenium like android native app automation framework with record and replay features.
The tool have three different components:

  • Server - Records user actions through Recorder and allows users to export their test-cases for use with the Runner.
  • Recorder - Tracks user actions at the android app side and then send them to the Server.
  • Runner - Allows the user to run their recorded test-cases on the actual android application as a suite. It also generats emailable html reports at the end of test execution.

This tool is based on keyword driven approach. For defining a test-case user have to define test-cases in "csv" format. These csv files will be converted to java files and then executed over the test app based on the configured execution framework. It uses following open-source tools:

1. Robotium
2. Selenium Nativedriver
3. TestNG
