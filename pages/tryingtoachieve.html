Whats bot-bot is trying to achieve

Currently bot-bot is trying to achieve the status of an open-source automation framework for native android application automation. We trying to achieve the following features: 1. Recording user interactions from an android native app. 2. Support converting the recorded cases to test-files to be used with bot-bot. 3. Support for addition/modification of steps by a user for a recorded step and then reverifying them by running it. 4. Support for suggestions for identifying elements based on different criterias.
